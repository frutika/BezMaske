# Project Overview

A full-stack web application featuring a React frontend, Express.js backend API, and PocketBase for database and authentication.

## Tech Stack

- **Frontend:** React 18, Vite, TailwindCSS, shadcn/ui, Framer Motion
- **Backend:** Express.js (Node.js)
- **Database & Auth:** PocketBase
- **Infrastructure:** Docker, Docker Compose, Nginx
